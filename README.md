# 🎲 D&D Helper – Калькулятор урона и симулятор кубиков

**Программа для игроков и мастеров Dungeons & Dragons 5-й редакции**, которая помогает:
- Рассчитывать **ожидаемый урон за раунд (DPR)** с учётом вероятности попадания, бонусов, критических ударов и режимов «преимущество»/«помеха»;
- Строить **графики DPR** в зависимости от класса брони (КД) цели и сравнивать разные оружия/заклинания;
- **Симулировать броски кубиков** с произвольным бонусом.

Проект разрабатывается в рамках вступительного кейса «Инженерный прототип для решения личной задачи» (Бакалавриат «Технологии искусственного и дополненного интеллекта»).

---

## 📌 Проблема и целевая аудитория

В D&D 5e важно балансировать урон персонажей, чтобы бои были интересными, не слишком лёгкими, но и не смертельными. Игроки часто подбирают оружие и заклинания «на глаз», а мастерам сложно оценить, сколько урона способен нанести монстр.

**Программа решает эту проблему** за счёт точного математического расчёта среднего урона за раунд (DPR) и позволяет сравнивать разные билды или тактики.

**Для кого:**  
- Игроки, которые хотят оптимизировать свои атаки.  
- Мастера, проверяющие сложность будущих сражений.

## 🚀 Как запустить

1. Убедитесь, что установлен Python 3.10 или новее.
2. Откройте терминал:
    - Нажмите сочетание клавиш Win + r
    - В появившемся окне введите cmd
    - Появится окно с терминалом
3. Склонируйте репозиторий:
    ``` bash 
   git clone https://github.com/StepanSorokin1/DnD-helper.git
    ```
4. Перейдите в папку с проектом:
    ``` bash
    cd DnD-helper
    ```
5. Установите зависимости:
    ``` bash 
    pip install -r requirements.txt
    ```
6. Запустите программу:
    ``` bash
    python main.py
    ```
7. Хорошей игры в ДнД!

## 🖥️ Пример работы
После запуска в терминале откроется главное меню:

```
=== D&D Helper ===
1. Быстрый расчет DPR
2. График DPR
3. Симуляция костей
0. Выход
```

### 1. Быстрый расчёт DPR
Вы вводите параметры атаки, программа выдаёт средний урон за одну атаку.

Пример:
```
Введите КД: 15
Введите бонус к попаданию, например '0', '+1' или '-2': +6
Введите бросок костей в формате 'количествоkгранность', например '2k6': 1k8
Введите бонус к урону, например '0', '+1' или '-2': +4
Режим (adv/dis/no): normal

Ваш DPR = 6.25
```

Режимы:
- no – обычный бросок
- adv – с преимуществом (advantage)
- dis – с помехой (disadvantage)

### 2. График DPR
Вы вводите параметры атаки, а программа строит столбчатую диаграмму зависимости DPR от КД цели (от 1 до 30).
На графике выделяется столбец, соответствующий введённому вами КД.

Дополнительно можно сравнить два оружия: после ввода первого вам предложат добавить второе, и тогда будет построен линейный график с двумя кривыми.

Пример:
```
Введите КД: 15
... (первое оружие)
Добавить еще одно оружие для сравнения? (y/n): y
Введите КД: 15
... (второе оружие)
```

Появится окно с графиком.

### 3. Симуляция костей
Вы указываете кость (например, 2k6) и бонус, программа выдаёт список выпавших значений и их сумму с учётом бонуса.

Пример:
```
Введите бросок костей в формате 'количествоkгранность', например '2k6': 2k8
Введите бонус к броску, например '0', '+1' или '-2': -1
Значения: 4 6
Сумма: 9
```
